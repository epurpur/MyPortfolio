Portfolio

OneFigr App

The OneFigr App analyzes and visualizes collection use metrics for the University of Virgina Libraries organization. This is a python web application built on the Django framework. The app analyzes datasets in various ways using the python libraries Pandas and NumPy before returning the data to the front-end. The user interface allows users to modify their view of the data on the fly and displays it using HighCharts, a javascript plotting library.

EcoValuator Plugin

This is a python plugin built for the Open Source GIS software, QGIS. EcoValuator was built for Key-Log Economics to be used as a simple means of analyzing land valuation throughout North America. Using the QGIS Python API (PyQGIS), the plugin takes publicly available land use/land cover data, analyzes it, and returns a view of land value by various parameters in the user's study area. It is freely available for download by any QGIS user.

Boone Real Estate

Since 2018 I've collaborated with Boone Real Estate to improve and manage a direct-mail marketing campaign. I set up a PostgreSQL database and used python scripts as well as PostgreSQL's PostGIS spatial extension in order to clean the large dataset (~230k records). I later integrated this data into the company's Salesforce instance and configured Geopointe, a native Salesforce mapping extension, to visualize the data spatially. This allows for easier integration with the company's direct-mail workfow and management of its internal data.

Light Cville

A streetlight network is a large investment in resources and infrastructure for any city. The City of Charlottesville, Virginia was looking for an improved way for city employees to manage its streetlight utilities. Light Cville is a full-stack Node application built on the MVC Framework. It has a MySQL database and Sequelize ORM, Express server and restful API, and Handlebars templating engine for server side rendering. The front-end utilizes the Leaflet mapping library and Tabulator to render the data table.

Is It Dry?

Is It Dry? is my original programming passion project. As an avid rock climber, I want to have the best experience possible while I'm out on the rocks. This project has evolved over time into a python Flask application that allows users to check the weather conditions at around 500 climbing locations globally using weather data from OpenWeatherMap. The map is implemented with Google Maps Javascript API and location data is stored in a lightweight SQLite back end.

National Park Finder

National Park Service finder is a simple front-end web application built to allow searching of sites administered by the National Park Service. This includes 466 sites including National Parks, National Historic Sites, National Monuments, etc. The site uses Google's Materialize CSS, Leaflet javascript mapping library, and jQuery UI for some of the UI components. All data is fetched from the National Park Service API.
